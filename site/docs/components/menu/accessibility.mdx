### Interactions

- Users reveal a menu by activating a focusable UI element, like a menu button.
- Users can close a menu by pressing Escape or clicking outside of it.

### Keyboard interactions

<KeyboardControls>
<KeyboardControl keyOrCombos="Tab or Shift + Tab">

- Closes menu and moves focus to the next (Tab) or previous (Shift + Tab) item in the tab order.

</KeyboardControl>
<KeyboardControl keyOrCombos="Enter or Space">

- Opens the menu and places focus on the first menu item.
- When focus is on a menu item that has a submenu, opens the submenu and places focus on the first item.
- Otherwise, activates the item, which is equivalent to activating the link element from which the menu item is made, and closes the menu.

</KeyboardControl>
<KeyboardControl keyOrCombos="Up arrow">

- When focus is in a menu, moves focus to the previous item.
- When focus is on a menu item that has a submenu, opens the submenu and places focus on the last item in the submenu.

</KeyboardControl>
<KeyboardControl keyOrCombos="Down arrow">

- When focus is in a menu, moves focus to the next item.
- When focus is on a menu item and that menu item has a submenu, opens the submenu and places focus on the first item in the submenu.

</KeyboardControl>
</KeyboardControls>
